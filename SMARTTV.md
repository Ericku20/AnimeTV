# 📺 Guía de Navegación Smart TV

Instrucciones detalladas para usar AnimeTV en Smart TV y televisores Android.

## 🎮 Controles Disponibles

### Flechas Direccionales
| Tecla | Función |
|-------|---------|
| ↑ | Moverse hacia arriba en el grid |
| ↓ | Moverse hacia abajo en el grid |
| ← | Moverse hacia la izquierda |
| → | Moverse hacia la derecha |

### Selección
| Tecla | Función |
|-------|---------|
| Enter / OK | Seleccionar elemento enfocado |
| Return | Seleccionar elemento |
| Space | Seleccionar (en algunos controles) |

### Navegación
| Tecla | Función |
|-------|---------|
| Escape | Volver atrás / Cerrar menú |
| Back | Volver atrás |
| Home | Ir al inicio |

---

## 🎯 Navegación en la Home

1. **Elemento inicial**: Primera tarjeta de la sección "Popular"
2. **Navegación**: Usa flechas para moverte entre tarjetas
3. **Cambiar sección**: Presiona ↓ para ir a la siguiente sección
4. **Ver anime**: Presiona Enter en la tarjeta deseada

### Layout de Grid
```
[Anime 1] [Anime 2] [Anime 3] [Anime 4]
[Anime 5] [Anime 6] [Anime 7] [Anime 8]
```

---

## 🔍 Búsqueda con Control Remoto

### Método 1: Búsqueda por Voz
1. Algunos controles remoto tienen micrófono
2. Di "buscar [nombre del anime]"
3. O presiona el botón de búsqueda en el control

### Método 2: Teclado Virtual
1. Navega al campo de búsqueda (arriba en el header)
2. Presiona Enter para activar
3. Usa el navegador de teclado:
   - Flechas para seleccionar letras
   - Enter para confirmar letra
   - Escape para limpiar

---

## ▶️ Reproductor de Video

### Controles del Reproductor
| Acción | Control |
|--------|---------|
| Play/Pause | Enter o Space |
| Avanzar 10s | → (en algunos reproductores) |
| Retroceder 10s | ← (en algunos reproductores) |
| Volumen arriba | ↑ |
| Volumen abajo | ↓ |
| Pantalla completa | OK |
| Salir de video | Escape o Back |

### Navegación en Menú de Configuración
1. Presiona Enter en el icono de configuración ⚙️
2. Usa flechas para navegar opciones
3. Presiona Enter para cambiar opción
4. Presiona Escape para cerrar

---

## 📱 Navegación en Página de Anime

### Seleccionar Episodio
1. En el sidebar derecho, se muestra la lista de episodios
2. Usa ↓ y ↑ para navegar entre episodios
3. Presiona Enter para reproducir

### Layout de Página
```
┌─────────────────────────────────┬──────────────┐
│  REPRODUCTOR DE VIDEO           │  EPISODIOS   │
│  (Arriba)                       │  ↓ Ep. 1     │
├─────────────────────────────────┤  Ep. 2       │
│  INFORMACIÓN DEL ANIME          │  Ep. 3       │
│  Título, sinopsis, géneros     │  ...         │
│  (Abajo)                        │              │
└─────────────────────────────────┴──────────────┘
```

## ❤️ Sistema de Favoritos

### Agregar a Favoritos
1. Navega a la tarjeta de anime
2. Presiona Enter para seleccionar
3. En la página del anime, busca el botón ❤️
4. Presiona Enter para agregar

### Ver Favoritos
1. Navega a "Favoritos" en el header
2. Presiona Enter para ir
3. Verás tu lista de favoritos

## 🎮 Consejos para Mejor Experiencia

### Enfoque Visual
- El elemento enfocado tiene:
  - **Ring rojo** alrededor
  - **Escala aumentada** (un poco más grande)
  - **Sombra destacada**

### Scroll Automático
- El reproductor automáticamente desplaza el contenido
- Mantén la vista en el elemento enfocado

### Navegación Inteligente
- El sistema busca el elemento más cercano
- Navega diagonalmente cuando no hay elemento arriba/abajo

## 🆘 Solución de Problemas

### No responde a controles
1. **Reinicia la página**: Presiona F5 o Back dos veces
2. **Verifica el navegador**: Usa Chrome, Firefox o Safari
3. **Recarga la app**: Cierra y abre nuevamente

### Busca no funciona
1. Asegúrate que el campo de búsqueda esté enfocado (ring rojo)
2. Intenta presionar Enter en el botón de búsqueda
3. Usa búsqueda por voz si está disponible

### Video no inicia
1. Comprueba la conexión a internet
2. Intenta con otra calidad en opciones
3. Recarga la página

### Control remoto no funciona
1. Verifica que la TV esté en el modo correcto (aplicación enfocada)
2. Algunos controles requieren emparejar primero
3. Intenta con teclado como alternativa

## 💡 Mejores Prácticas

1. **Posicionamiento**: Coloca la TV a una altura cómoda
2. **Distancia**: A 3-4 metros de la TV (según tamaño)
3. **Iluminación**: Ambiente semi-oscuro para mejor visualización
4. **Controles**: Aprende los botones principales de tu control
