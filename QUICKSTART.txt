🚀 1. INSTALACIÓN RÁPIDA
═══════════════════════════════════════════════════════════════════════════

Opción A - Script automático (RECOMENDADO):
────────────────────────────────────────
  bash setup.sh

Opción B - Manual:
──────────────────
  1. npm install
  2. cp backend/.env.example backend/.env.local
  3. cp frontend/.env.example frontend/.env.local
  4. npm run dev

✅ Verificación:
  - Frontend: http://localhost:3000
  - Backend:  http://localhost:3001

⚙️ 3. COMANDOS PRINCIPALES
═══════════════════════════════════════════════════════════════════════════

DESARROLLO:
───────────
  npm run dev              # Inicia Frontend + Backend
  npm run dev:frontend     # Solo frontend
  npm run dev:backend      # Solo backend

PRODUCCIÓN:
───────────
  npm run build            # Compila para producción
  npm start                # Inicia en producción

CALIDAD DE CÓDIGO:
──────────────────
  npm run lint             # Ejecuta linter
  npm run type-check       # Verifica tipos TypeScript
  npm run format           # Formatea código

🆘 7. SOPORTE Y CONTACTO
═══════════════════════════════════════════════════════════════════════════

¿PROBLEMAS?
──────────
  1. Lee TROUBLESHOOTING.md
  2. Limpia node_modules y reinstala
  3. Revisa la consola del navegador (F12)
  4. Verifica que backend esté corriendo

REPORTAR BUG:
─────────────
  1. Abre un issue en GitHub
  2. Incluye: OS, navegador, Node.js version
  3. Adjunta logs y capturas

SUGERIR MEJORA:
───────────────
  1. Abre una discussion
  2. Describe el caso de uso
  3. Explica qué problema resuelve

CONTRIBUIR:
───────────
  1. Lee CONTRIBUTING.md
  2. Fork el repositorio
  3. Crea una rama
  4. Haz cambios
  5. Crea Pull Request

📞 INFORMACIÓN RÁPIDA
═══════════════════════════════════════════════════════════════════════════

URLs:
  Frontend:   http://localhost:3000
  Backend:    http://localhost:3001
  Health:     http://localhost:3001/health

Puertos:
  3000  → Next.js (Frontend)
  3001  → Express (Backend)

Stack:
  Frontend:   Next.js 14 + React 18 + TypeScript + TailwindCSS
  Backend:    Node.js + Express + TypeScript
  Storage:    localStorage (sin backend)
  API:        Anime1v API proxy

Créditos:
  API de Anime1v por FxxMorgan
  Componentes con Framer Motion
  Estilos con TailwindCSS
