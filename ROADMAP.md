# 🗺️ Roadmap de AnimeTV

Visión futura y planes para AnimeTV.

## 📍 Estado Actual (v1.0.0)

### ✅ Implementado

**Core Features:**
- [x] Búsqueda de anime
- [x] Página de detalles del anime
- [x] Reproductor de video HLS
- [x] Sistema de favoritos (localStorage)
- [x] Continuar viendo
- [x] Historial de visualización
- [x] Navegación Smart TV con flechas
- [x] Responsivo (móvil, tablet, desktop, TV)
- [x] Modo oscuro por defecto
- [x] Carruseles horizontales
- [x] Búsqueda con debounce
- [x] Infinite scroll
- [x] Skeleton loaders

**Tech Stack:**
- [x] Next.js 14 + React 18
- [x] TypeScript
- [x] TailwindCSS
- [x] Zustand
- [x] hls.js
- [x] Framer Motion
- [x] Express Backend
- [x] Axios
