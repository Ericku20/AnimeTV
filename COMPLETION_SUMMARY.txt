💡 TIPS IMPORTANTES
═══════════════════════════════════════════════════════════════════════════

1. DESARROLLO:
   - Usa npm run dev para desarrollo rápido
   - Abre http://localhost:3000 en navegador
   - Consola del navegador (F12) para errores

2. SMART TV:
   - Usa flechas + Enter para navegar
   - Perfecto para Android TV, Google TV, etc.
   - Menú responsive en móvil

3. REPRODUCCIÓN:
   - VideoPlayer soporta HLS
   - Controles grandes y accesibles
   - Selector de calidad integrado

4. ALMACENAMIENTO:
   - Datos guardados en localStorage
   - Sincronización entre pestañas
   - Persiste entre sesiones
